Navigator Object & Screen Object Properties

Below are is a list of a few properties that the Navigator Object and the Screen Object use to gather information of the current browser app.

  • Navigator Object Property:appName Description-Name of the web browser displaying the page
  • Navigator Object Property:appVersion Description-Version of the web browser displaying the page
  • Navigator Object Property:geolocation Description-API for accesing the user's current location and user permission setting denying or allowing access to that information
  • Screen Object Property:availHeight Description-Height of the display screen
  • Screen Object Property:availWidth Description-Width of the display screen
  • Screen Object Property:pixelDepth Description-Display screen's color resolution in bits per pixel
